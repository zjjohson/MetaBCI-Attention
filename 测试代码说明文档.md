# 测试代码说明文档

**项目名称**：基于 MetaBCI 的学习注意力实时监测与反馈系统

**所属赛道**：被动监测

---

## 1. 测试环境

| 项目 | 要求 |
|---|---|
| Python | 3.10 |
| 操作系统 | Windows 10/11 |
| 硬件 | Neuracle 脑电帽（可选，支持模拟模式） |
| 依赖安装 | `pip install -r requirements.txt` |

---

## 3. 测试命令汇总

### 3.1 模拟模式测试（无需硬件）

```powershell
# 终端 1：启动设备模拟 + 实时推理
.venv310\Scripts\python.exe run_device.py --device simulate

# 终端 2：启动前端仪表盘
.venv310\Scripts\python.exe frontend\app.py --port 5000

# 浏览器打开 http://localhost:5000 查看仪表盘
```

**预期效果**：
- 模拟数据源生成随机 EEG 信号
- Socket 9999 端口发布解码状态
- 前端仪表盘实时渲染波形曲线与状态环

---

### 3.2 设备模式测试（需 Neuracle 脑电帽）

```powershell
# 终端 1：连接 Neuracle 设备 + TCN 实时推理
.venv310\Scripts\python.exe run_device.py --device neuracle_tcp --tcp-channels 3 --model tcn

# 终端 2：启动前端仪表盘
.venv310\Scripts\python.exe frontend\app.py --port 5000
```

**预期效果**：
- NeuracleTCPAdapter 连接 `127.0.0.1:8712` 读取 float32 数据流
- 提取 Fp1、Fp2 双通道，250Hz 采样率，1.0s 滑窗推理
- 前端仪表盘 SSE 流实时更新 α/β 波形与注意力状态环
- 端到端延迟 < 500ms

---

### 3.3 刺激界面测试

```powershell
# MEMA 协议刺激范式（需 Neuracle 脑电帽）这一部分需要目录下有一个media文件夹（视频太大没有上传到github，可自行准备）
.venv310\Scripts\python.exe brainstim\mema_protocol.py --subject 1 --blocks 1
```

**预期效果**：
- 依次呈现 Neutral（十字注视）→ Relaxing（风景视频+音乐）→ Concentrating（学习视频）三阶段
- 每阶段结束后弹出注意力自评 + VAD 情绪量表
- 刺激阶段标记写入 EEG 数据文件

---

### 3.4 TCN-Attention 模型训练测试

```powershell
# 训练 TCN-Attention 模型（n 轮）
.venv310\Scripts\python.exe train_tcn.py --epochs n
```

**预期效果**：
- 加载 MEMA 数据集（27,360 段）+ 个人校准数据（1,639 段）
- 输出测试集分类报告（准确率、召回率、F1）与混淆矩阵
- 目标准确率 ≥ 78%
- 生成 `models/tcn_attention_best.pt` 和 `models/tcn_config.json`

---

### 3.5 个人数据采集 + MLP 混合训练

```powershell
.venv310\Scripts\python.exe collect_real_data.py --train
```

**预期效果**：
- 采集个人校准数据（Fp1/Fp2 双通道，250Hz）
- 训练 MLP 分类器，保存 `models/real_classifier.pkl` 与 `models/real_scaler.pkl`

---

### 3.6 MEMA 数据集预处理

```powershell
.venv310\Scripts\python.exe download_mema.py
```

**前提**：先从 [百度网盘](https://pan.baidu.com/s/1ssvZWAI6gwV2ey0cRogDWg?pwd=2dg7) 下载 `For_DL` 目录，解压至 `attention_dataset/mema/For_DL/`

**预期效果**：
- 生成 27,360 段预处理数据，每段形状 (2, 250) @250Hz
- 三类标签（涣散/一般/集中）各 9,120 段，均衡分布
- 生成 `meta.json` 标签映射文件
